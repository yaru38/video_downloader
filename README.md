## 概要

Youtubeからmp4、mp3をダウンロードするスクリプトです。

本スクリプトはほぼOpen AIが提供するcodexによって作成されています。よって、投稿者にコードの説明を求めてるのはご遠慮ください。


## 使用方法

- pythonを実行してGUIを開く
- 上部のテキストボックスにDLしたい動画のURLを入力 ※1
- MP3、MP4から選択してDL

## 使用上の注意

使用はすべて自己責任で行ってください。

※1 再生リストのURLは使えません。ミックスリストなどからURLを取得するとうまく作動しないので、動画の共有ボタンからURLを取得してください。
